# Student Management System

A simple web-based Student Management System developed using PHP and MySQL.

## Features
- Add student details
- View student list
- Delete student records
- Responsive UI using CSS

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP

## How to Run
1. Install XAMPP
2. Place project folder in `htdocs`
3. Create database and table using SQL
4. Run `http://localhost/student_management`
